# Technical Interview Repository

This repository contains interview materials: prompts, starter code, reference solutions, and evaluation notes.

## Stack

TypeScript · Node.js · React · Database (SQL/ORM)

## Structure

Materials are organized by date and topic (e.g. `260320-Typescript-Node-React-Database`).

## Working here

- Prefer editing existing files over creating new ones
- Keep solutions minimal — interview code should be readable at a glance
- Do not add extra dependencies beyond what the prompt specifies
